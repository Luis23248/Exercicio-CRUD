# Exercicio-CRUD

### Objetivo:
1. Realizar um CRUD.
- Create (Criar)
- Read (Buscar)
- Update (Atualizar)
- Delete (Excluir)

2. Utilizar um banco de dados relacional
- MySQL

### Tecnologia:
- Linguagem de programação: Python
- ORM: SQLalchemy
- Banco de dados: MySQL
- Containerização: Docker
- Orquestração de conteineres: Docker Compose
- Versionamento: Git
